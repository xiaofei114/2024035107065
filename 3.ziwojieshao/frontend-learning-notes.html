三件套对比表

技术 主要职责 生动比喻 常见用法
HTML 定义网页的结构和内容 房子的框架和地基 创建标题、段落、图片、链接等
CSS 控制网页的外观和布局 房子的装修和装饰 设置颜色、字体、间距、动画等
JavaScript 实现网页的交互和动态效果 房子的智能控制系统 表单验证、动画效果、数据交互等

5个最常用的HTML标签

<div> - 容器标签
最常用的容器标签，用于组织和布局页面内容。它是一个块级元素，可以包含其他元素。
<div>这是一个容器</div>
<p> - 段落标签
用于定义文本段落，浏览器会自动在段落前后添加一些空白。
<p>这是一个段落</p>
<a> - 链接标签
用于创建超链接，可以链接到其他页面、文件或同一页面的其他位置。
<a href="https://example.com">点击这里</a>
<img> - 图片标签
用于在网页中嵌入图片，需要指定图片的来源路径。
<img src="image.jpg" alt="图片描述">
<h1> ~ <h6> - 标题标签
用于定义标题，h1是最大的标题，h6是最小的标题。用于构建文档的层次结构。
<h1>这是一级标题</h1>
